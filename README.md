# Zhalae Daneshvari Personal Website

Personal website built with React and Vite.

## Live Site

- https://zhalaedaneshvari.github.io/zhalae-website/


## Why I Built It This Way

I wanted a portfolio that feels both polished and human.

- Professional enough for recruiting and technical reviews
- Personal enough to sound like me, not like a generic template
- Fast enough to load quickly and stay smooth

The homepage highlights my focus areas in data science, machine learning, UX, agentic AI, and biotech-oriented technology.

## Tech Stack

- React 18
- Vite 5
- Plain CSS with design tokens and motion-safe fallbacks

## Optimization Choices

- Lightweight frontend architecture with no unnecessary runtime libraries
- CSS variable theme system for efficient light/dark switching
- `prefers-reduced-motion` support for accessibility
- Minimal JavaScript for animation (typing, section reveal, role rotation)
- Vite build pipeline for fast local development and optimized production output

## Local Development

```bash
npm install
npm run dev
```

Open the local URL shown in terminal (usually `http://localhost:5173`).

## Production Build

```bash
npm run build
npm run preview
```

## Deployment

This project is deployed with GitHub Pages using a GitHub Actions workflow.

- Workflow file: `.github/workflows/deploy-pages.yml`
- Deploy trigger: push to `main`
